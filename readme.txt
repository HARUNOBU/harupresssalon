== Description ==

This theme is a theme for therapists such as salons such as aroma salon relaxation salon

Meshie uses the Bootstrap base of the CSS framework and it is created based on _s of the starter theme.

=== Features ===
Responsive Layout, Theme Customize, Page Navigation

=== Global navigation menu ===
Global navigation menu is displayed in the header.

=== Image of archive page ===
In the article list page (archive page) of the blog, thumbnails are usually displayed as images of eye catch images.

=== Theme customizer ===
Theme customizer is available. The recommended size for the header image is width 980px height 350px.

== Installation ==

=== Installation using "Add New Theme" ===
1. From your Admin UI (Dashboard), go to Appearance => Themes. Click the "Add New" button.
2. Search for theme, or click the "Upload" button, and then select the theme you want to install.
3. Click the "Install Now" button.

=== Manual installation ===
1. Upload the "chocolat" folder to the "/wp-content/themes/" directory.

=== Activiation and Use ===
1. Activate the Theme through the "Themes" menu in WordPress.
2. Go to Appearance => "Meshie Settings" page. Complete the input of all necessary, please click on the "Save changes" button.
